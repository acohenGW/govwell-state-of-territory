# govwell-state-of-territory

A reporting tool that produces a **State of Territory** — a coverage analysis showing how much of GovWell's Ideal Customer Profile (ICP) has been contacted in each sales territory.

## Purpose

Sales and RevOps teams use this repo to answer the question:

> *How well are we covering our addressable market?*

For each territory, the tool measures what percentage of ICP accounts have been contacted (called, emailed, or otherwise engaged), making it easy to spot under-served areas and prioritize outreach.

## What it produces

- Per-territory coverage metrics (% of ICP contacted)
- Breakdowns by rep, region, or any territory dimension
- Gaps analysis — ICP accounts with zero contact activity
- A rollup view across all territories

## Data sources

The tool pulls from the sources needed to join ICP account data with contact activity (e.g. CRM exports, enrichment data). See the configuration section once data connectors are added.
